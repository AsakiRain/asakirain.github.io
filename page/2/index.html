盼望着，盼望着，考试来了，死神的脚步近了。一切都像刚死去的样子，凄凄凉流下了泪。作业多起来了，试卷堆起来了，我们的脸囧起来了。

老师偷偷的从办公室里走出来，凶神恶煞的。教室里、操场上瞧去，一大片一大片满是试卷。做着，写着，涂两下修正液，打两个草稿，风哗哗吹的，草摇来摇去的。语文，数学，英语，你不让我，我不让你，都发了试卷赶趟儿。多得像雨，堆得像山，做得像鬼。卷里带着些难高兴起来了，呼朋引伴的***，准备报复老师。他们的头发稀稀疏疏的，在雨里飘荡着。

天上的冤魂渐渐多了，地上的坟墓也多了。城里乡下，家家户户，老老小小，也赶趟似的，一个个都出来了。祭奠祭奠孩子，焚烧焚烧试卷。“考试害死多少学生”，刚起头儿，以后有的是受罪的孩子，有的是破灭的希望。

考试像刚出世的灾星，从头到脚都是坏的，它破坏着。考试像游荡的死神，面目狰狞的，奸笑着奔跑着。考试像恐怖的噩梦，有铁一般的胳膊和腰脚，把我们带往地狱。

今天（周六）回家，无意间看到taptap的通知，才知道食物语ios的删档内测开启了。很幸运，我在我的邮箱里找到了testflight的内测资格码。于是很开心地下载之。

700MB的游戏，大小还算适中。进入登陆页面后如期出现了腾讯传统：“与QQ好友玩、与微信好友玩和游客登陆。”emmm。好吧，QQ登陆！值得一提的是，本来登录界面UI是有动效的，不过首次加载的时候我没显示，不知道是不是bug呢。

游戏开头就有一段有趣的起床剧情（哇佛跳墙美如画）和世界观介绍，我挺喜欢这样的设定。佛跳墙的片段。从这张cg里看出好像是异色瞳。但是在live2d里好像没体现出来。

不过自然，任何一个游戏都不会让你有一个好的开局，不久之后，主角就被迫毁坏了食物语，切断了与食魂之间的记忆羁绊，这里剧情处理得很好，不显得尴尬，而是给人刻下一种重新寻回旧日时光与记忆的执念。

北京烤鸭手的动效

食魂们自然都很好看，像开头剧情里的锅包肉、佛跳墙或者是北京烤鸭我都很喜欢ww（顺带一提，我喜欢好看的男孩子呀）live2d做的很精细，以前我玩过梦间集，但是完美做的live2d并没有人物手上的动作。同时，食物语角色们的配音都很齐全。（可惜我在返校之前没一个能把他们寻回的）

看完了《付丧神出租中》

发表于 2019-05-11 更新于 2025-12-08 分类于 媒体 ， ACG

图片取自萌娘百科

这周把这部番剧看完了。

这部番总体来说还是令我十分满意的。我个人比较喜欢与日系妖怪或是神明有关的题材，所以说这部番恰好和我胃口。

不过值得一提的是，我的喜好可能是比较小众的，因为看这部番的人少之又少，并且我曾经看过的番剧也有身边人闻所未闻，比如《结界师》、《钢铁神兵》这类的。

说回这部番剧。故事围绕出云屋店主阿红和清次而一个个展开，几件事情结合紧凑，先前出场，并解决了问题的角色在之后还会频繁出现，给人以一种解决问题顺便交上朋友的有趣感觉。

被叫做名侦探的清次破解问题的思路十分的清晰，其中部分的推断我自己也有所思考并和剧情吻合，不像看《名侦探柯南》一样（说实话这个没啥可以相比的）一脸懵逼就听到柯南说：“我知道了”，然后傻傻地听着他陈述犯罪手法。

在这其中，付丧神们也帮了大忙。我最初知道付丧神是在刀剑乱舞里，这部番中的付丧神更贴合我的想象，毕竟由物品化身而来，他们的特征都十分像本体。每一个付丧神的形象都很鲜明，5个付丧神里没一个摸鱼的，也没一个是配角，所有人都有发挥恰到好处的作用。所以说刻画地很鲜明成功。

我挺喜欢番剧的OP和ED的。——是的，很少有番剧能让我同时喜欢上片头片尾曲，曲子我暂时还没办法放出。简单来说，OP极富动感，令人耳目一新，ED声音和画面配合的很好，给人以空灵美好的感觉，仓木麻衣的声音真的挺好听的，而且歌词写的也挺好，紧扣住了结局。

整部番剧中，我最喜欢的角色是半助先生了。在他的身上，我看到了一个江户女子在遭遇感情破裂后，以一种男扮女装的全新方式重现在人们眼前，并且取得大家的认可的奇特经历，这使我耳目一新的。——并且，我觉得他好看ww。

在番剧的结尾，暗恋了整部番剧阿红的清次终于在阿红的主动操作下有了在一起发糖的机会，这样的情节设计我很喜欢，因为这是HappyEnd，并且两个人解决了之前主人公们的一系列问题（多为情感问题）后，终于解决了他们两个人自己的感情问题XD

摘自最终话下方的评论：“终于可以说了（。）原作小说的续作讲的是他俩孩子的故事，无论动画改不改世界线会收束的[2233娘_卖萌]”所以说这一段有趣的故事算是结束了，可能是我内心感知的影响，我并不想去找续作来看，所以就让这个故事的圆满结局一直停留在番剧结尾好了。

记宁中百廿校庆

发表于 2019-05-11 更新于 2021-08-16 分类于 日常

今天周六，在学校里待了一天，参加了整场宁波中学的校庆。

总体来说我还是挺高兴的，不仅仅只是别人眼里说的“120周年校庆难得一遇，你们很幸运”，而且还有活动带给我的充实感。

我喜欢今天上午的表演。就拿以前七中的校庆来对比，总感觉它更胜一筹。各种重要活动穿插在表演中，表演是紧贴校庆主题的，而不是像七中那样先是各种致辞，完了以后就只有表演。同时，会场上的几块大屏幕也给表演增色不少，因为一部分的表演通过屏幕能更好地传递情感。这些屏幕校庆三天前就开始一块块拼起来了，最后开完上午的典礼就拆了，让我感觉有点可惜——不过这是在正常不过的操作嘛。在上午的庆典活动中，我确实有被感染到。具体怎样就不多说了。

下午我们听了贺贤土院士关于他的核研究经历的讲座——说是讲座，我其实觉得更像是交流会，因为贺先生在讲座中并没有像演讲一样提出鲜明的呼吁或者号召，只是在用心地讲述他的核武器研究历程。我喜欢这样的讲座，因为我觉得富含自己亲身经历的讲座能让人从心里有所领会或者感想，而不是一味地说理，或是举一些似真非真的例子，只是让人费力地理解，听完以后什么都没有记住。贺先生的经历让我详细地了解了中国是怎么研究出原子弹，再是氢弹，继而是现在正在着手研发的中子弹。在他结束讲述后的提问也让我认识到一些有趣的内幕——比如说中国和美国原子弹的结构并没有媒体宣传的那样有所不同，而是基本一致的，这一点具贺先生所说是没有根据，只是媒体炒作，没有经过辟谣的。

不过，今天的校庆也有一些小事情让我觉得有点不舒服。

上午在操场看表演的时候，到了唱歌、跳舞这些可能有本班同学或者认识的老师参与的节目，前面有一批人站起来或者是半蹲着认人，其中不乏我们班一部分的同学。基本算是坐在班级末尾，前面的同学这一系列在我看来缺乏素质的举动使我完全看不见台上的任何表演。同时，身后有同学不停地讲话讨论游戏，乐此不彼。虽然他也有注意控制音量，但是我还是觉得不妥。

下午听讲座，当来到了同学提问环节时，全场都有些躁动，加之贺先生此时声音不再那么响亮，有时候身边的同学一吵，就听不清楚他在讲什么。

——之所以把坏的说在后头，是因为在我的心中，这次的校庆是非常成功的，这一些小事情也不是有必要搬到台面上讲（好好批斗一番），加之我也认识到，我的三观和班里一些同学本来就有一些不同。我在这种情况下应当是选择保持沉默，只是在心里不认同就好了。

看完了花丸 -續-

发表于 2019-05-11 更新于 2025-12-08 分类于 ACG

昨天晚上看完了花丸 -續- （下称花丸2）。

好开心啊，本来的话花丸2前面大部分是今年六月保送考之后看的。直到一个星期前，一直留着三集没有看。最后几集剧情稍稍波折了一下，不过整体来说还是很轻松圆满的。

——正如别的同事说的：1.花丸日常向

2.花丸是用来满足审神者各种美好的臆想的XD

安定在最后一集回来了，不过回来的历程很奇妙啊。

时至夕阳西下，安定在某个时代的田垄上面独自行走，身上的传送器突然发了光。

定睛一看，安定发现自己回到了这个本丸的最初时光，站在那棵熟悉的千叶樱下面。——千叶樱此时并没有开花。此时的本丸仅有加州清光，压切长谷部 ，今剑三振刀。清光在走廊的转角偷偷张望，看到今剑问起清光刀友（不吐槽这个名字了，字面意思就好）的事，清光自然提起了安定。安定转头跑回树旁，想与同一时间线上的清光相逢。——他用手抚着树干，嘴里默念了一句，让我回到清光身边。

此时的清光正在7-1战斗，面对对面的一群极化胁差，情况紧张。安定突然从天空中掉了下来。有幸突然重逢的两人士气高涨，双双真剑必杀并秀出了二打开眼的神奇操作。在所有人的合力战斗下，时间溯行军被打败。但是安定发现自己的传送器在战斗中遭到损坏。安定在清光一行人走之后不久也成功启动了传送器。然而在本丸的大家的眼里，安定在4个月后才回来。

所以说安定少活了4个月。这对于刀剑付丧神来说是什么概念那我也说不清楚啊XD。

花丸系列我觉得应该还会有第三季。不过现在放在眼前没看完的还有活击-刀剑乱舞这部番。

它讲的就是贴近历史的战斗方面的故事了，之前看过3集，感觉画面有点压抑，然后兼桑和堀川的心理都非常的矛盾，后面听说还有比较虐心的部分。所以还没准备看。

话说回来，我可不是一个称职的审神者。日服游戏已经弃了两个月没开了，国服快要半年没动过了。（刚刚想进日服发现在维护）而且肝游戏没什么耐心，——虽说刀剑乱舞游戏性确实不高，还是番剧好看
